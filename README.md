# Hotel Management System

This Hotel Management System is designed to simplify daily hotel operations, including room management, bookings, customer management, billing, and reporting. The system is built using Flutter and Firebase, following Clean Architecture and MVVM principles for maintainable and scalable code.

---

## Key Features

- **Rooms Management:**  
  - Add, edit, and delete rooms.  
  - Track room status (available, booked, under maintenance).  
  - Categorize rooms by type and price.

- **Booking Management:**  
  - Create new bookings for customers.  
  - Track current and past bookings.  
  - Notifications to remind staff about upcoming bookings.

- **Customer Management:**  
  - Add and edit customer information.  
  - View customer history and past bookings.

- **Billing & Payments:**  
  - Generate automatic invoices during booking or service registration.  
  - Export invoices to PDF and Excel.  
  - Open files on different devices (e.g., Excel or Notepad on Windows).

- **Reports & Analytics:**  
  - Daily, weekly, and monthly reports on bookings and payments.  
  - Export reports in various formats.

- **Automatic Updates:**  
  - Real-time data updates using Firebase Streams.  
  - Scheduled daily updates for important data.

---

## Technologies Used

- **Flutter:** Cross-platform UI development (Desktop & Mobile).  
- **Firebase Firestore:** Cloud-based real-time database.  
- **PDF & Excel Packages:** Generate invoices and reports.  
- **Clean Architecture + MVVM:** Organized and maintainable code structure.  
- **Cubit / Bloc:** State management for predictable app behavior.
